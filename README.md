### Requirements

- Python 3.5
- [PyTorch 0.4.0.](https://pytorch.org/previous-versions/) (GPU)

### Install Python dependency

```sh
pip install -r requirements.txt
```


## Usage


### Run experiments

```sh
./run.sh WORK_DIR DATANAME GPU_ID
```
